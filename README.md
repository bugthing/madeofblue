# MadeOfBlue

This repository is built from a template for building my own custom [bootc](https://github.com/bootc-dev/bootc) image with customizations to from the Universal Blue Project.

## Why?

I want a stable OS that I can customise to my liking.

## Development

Push changes to GitHub and Actions with generate a release in the container registery, you can use it like so:


      just run-vm-qcow2 ghcr.io/bugthing/madeofblue latest
